# CashFlowin

Aplikasi web untuk mengelola **pemasukan** dan **pengeluaran** keuangan. Dibangun menggunakan **Next.js + Firebase Firestore** dengan dukungan export data ke **Word (.docx)**.

- ‚úÖ Export laporan keuangan ke **Microsoft Word (.docx)**.
- ‚úÖ Perhitungan otomatis **saldo akhir**.
- ‚úÖ UI modern dengan **shadcn/ui** dan animasi dari **Framer Motion**.

## üöÄ Instalasi

1. **Clone repository**

```bash
   git clone https://github.com/azkbrqlna/CashFlowin.git
   cd CashFlowin
```

2. **Install Dependencies**

```bash
   npm install
   atau
   yarn install
```

3. **Setup Environment Variables**

```bash
    NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
    NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
    NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
    NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
    NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
    NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id
```
